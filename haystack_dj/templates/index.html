Searchly Sample Django Application.


This example illustrates basic search features of Searchly (Search as a service powered by Elastisearch).

Each CRUD operation on documents is reflected to search index in real time.

To test Searchly's search features login to admin screen and create new documents.

Sample application is using Haystack moduler search for Django.
